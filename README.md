# 🐦 HARPIA QUANTUM LABS - V18.2 AKASHIC

This repository contains the **Harpia V18.2** quantum simulation core, focused on high-performance vectorized processing and geometric visualization of quantum hydrogen networks within a Torus topology.

## 💎 Features

* **Akashic Core:** Single-pass memory processing designed to eliminate data fragmentation and maximize simulation speed.
* **Cloud Handshake (Kill-Switch):** Enhanced security protocol. The simulator requires an active validation handshake from the SPHY API to initiate processing.
* **Ethereal VR Engine:** Optimized graphical engine for real-time visualization of quantum coherence and systemic fidelity.
* **Vectorized Oracle:** Native integration with **PennyLane** for flow calibration using RZ and Hadamard gates.

## 🛠️ Installation

1. **Clone the repository or navigate to the folder:**
```bash
cd ~/QLZ_SDKS/PennyLane/toro

```


2. **Activate your virtual environment:**
```bash
source venv/bin/activate

```


3. **Install dependencies:**
```bash
pip install -r requirements.txt

```

## 🚀 Execution

The engine performs a **security handshake** before starting calculations. Ensure your internet connection is active so it can reach the API.

**To run the English version:**

```bash
python3 sphy_fibonacci_ai_eng.py

```

**To run the Portuguese version:**

```bash
python3 sphy_fibonacci_ai_pt-Br.py

```
